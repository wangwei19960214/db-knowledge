# 有货率报表生成流程

## 适用场景

- Wayfair 上架 SKU、链接 SKU、采购 SKU 的指定时间段有货率统计。
- 需要输出上架 SKU 有货率、链接 SKU 有货率、断货天数、缺失库存天数和异常明细。
- 可作为后续指定月份、指定店铺、指定链接 SKU 的复用报表模板。

## 输入参数

- 时间范围：必填，例如 `2026-05-01` 至 `2026-06-15`。
- 店铺 / 国家：必填或可由店铺映射，如 `WF-TD`、`WF-FOP` 使用美国仓，`WF-TD-CA`、`WF-FOP-CA` 使用加拿大仓，`WF-TD-UK` 使用英国仓。
- SKU 范围：可选，上架 SKU、链接 SKU、平台 SKU、采购 SKU。
- 输出形式：Excel、CSV、JSON、Markdown 摘要。
- 是否关联广告：可选，按店铺 + 平台 SKU 输出统计期广告 ID 和广告名称。

## 指标口径

- 上架 SKU 粒度为“店铺 + 上架 SKU”。
- 上架日计入，下架日不计入。
- 采购 SKU 优先使用 `wayfair上架sku销售映射`，缺失时用 `平台产品详情表.采购sku` 拆分补缺。
- 库存按采购 SKU、日期、店铺对应国家仓库汇总 `可用`。
- 当天无库存记录时向前取最近有效库存；此前从未出现则标记缺失。
- 组合产品任一采购 SKU 当天可用小于等于 0，即判定该上架 SKU 当天断货；全部组件大于 0 才算有货。
- 上架 SKU 有货率 = 有货天数 / 实际在架天数。
- 存在缺失库存日时，严格有货率留空，另提供已知日有货率用于排查。
- 链接 SKU 按“店铺 + 链接 SKU”聚合；空值、`NULL`、`无` 不作为有效链接。
- 上架 SKU 销量权重为截至统计截止日向前最近 90 个有货日的平均销量。
- 链接 SKU 每日有货率 = `Σ(上架SKU销量权重 × 当天有货状态) / Σ(当天实际在架上架SKU销量权重)`。
- 某链接当天有效销量权重总和为 0 时，使用等权回退。

## 执行步骤

1. 确认时间范围、店铺/国家、SKU 范围和输出格式。
2. 搜索并确认有货率、断货天数和 SKU 映射口径。
3. 读取在架商品，按上架/下架日期展开每日在架状态。
4. 建立上架 SKU -> 采购 SKU 映射，记录缺失和一对多。
5. 按店铺映射国家仓库，汇总采购 SKU 每日 `可用`。
6. 对缺失库存日向前取最近有效库存；无法补齐则记录缺失。
7. 按组合产品规则计算每日上架 SKU 有货状态。
8. 汇总上架 SKU 有货天数、断货天数、实际在架天数和有货率。
9. 按链接 SKU 聚合，使用销量权重计算链接 SKU 有货率。
10. 可选关联广告记录，输出广告 ID 和广告名称。
11. 输出异常清单和数据截止日期说明。
12. 保存产物到 `reports/`，更新闭环文档。

## 校验方式

- 检查库存、销量、广告数据各自日期覆盖和截止日期。
- 检查上架 SKU、链接 SKU、采购 SKU 映射是否缺失或一对多。
- 抽样核对组合产品组件库存与最终有货状态。
- 检查实际在架天数是否正确排除下架日。
- 检查严格有货率和已知日有货率是否按缺失库存日区分。
- 检查链接 SKU 权重为 0 时是否有等权回退。

## 输出产物

- Excel 主报表。
- 中间 JSON。
- 异常明细：无采购 SKU、无有效链接 SKU、链接 SKU 冲突、采购 SKU 从未出现库存、缺失库存日。
- 数据截止日期说明：库存、销量、广告分别列出。
- 抽样校验说明。

## 常见坑点

- 库存、销量、广告数据截止日期可能不同。
- 采购 SKU、链接 SKU 映射异常会影响有货率分母。
- 组合产品不能只看单个组件库存。
- 缺失库存日不能直接当作有货或断货。
- 链接 SKU 为空、`NULL`、`无` 不能作为有效链接。
